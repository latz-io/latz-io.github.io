Fabian Wagner, Iason Papaioannou, Elisabeth Ullmann and I have deposited the manuscript
["Multilevel Sequential Importance Sampling for Rare Event Estimation" in the arXiv](https://arxiv.org/abs/1909.07680).
The main idea is to use the Multilevel Sequential² Monte Carlo framework for reliability analysis rather than
Bayesian inversion; see [(2018, J. Comput. Phys. 368, p. 154-178)](https://doi.org/10.1016/j.jcp.2018.04.014).
In this manuscript, we derive a method based on this idea.
We show in numerical experiments that this is a resonable approach that can outperform single level strategies. 
Moreover, it solves nestedness issues that can occur in Subset Simulation algorithms.
Finally, we consider MCMC algorithms that are based on adaptive von Mises-Fischer-Nakagami proposals. Those seem to be especially
useful to use with Sequential Monte Carlo methods.
